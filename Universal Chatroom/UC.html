Universal Chat Room

Visit the live project: Universal Chat Room

Universal Chat Room is a 5-week web application project designed to support seamless, multilingual communication between users. Built with full database integration, the system stores messages, tracks sessions, and maintains chat history across user sessions. A built-in translation API allows users from different language backgrounds to communicate effortlessly in real time.
